2. Drop Shadow Filter

The drop-shadow() function creates shadows that follow the alpha contour of elements. In PDF, it's rendered as a box-shadow approximation.

drop-shadow(3px 3px 5px)

‚úÖ PDF Support: Drop shadows are rendered using the existing box-shadow infrastructure. Supports offset-x, offset-y, blur-radius, and color.

‚ö†Ô∏è Filters Not Rendered in PDF

The following filters are parsed correctly but not rendered in PDF (they require rasterization which is not yet implemented):

  • blur() - Gaussian blur effect
  • brightness() - Brightness adjustment
  • contrast() - Contrast adjustment
  • grayscale() - Convert to grayscale
  • sepia() - Sepia tone effect
  • saturate() - Saturation adjustment
  • hue-rotate() - Rotate hue values
  • invert() - Invert colors
  • backdrop-filter - All backdrop filter effects

‚ö†Ô∏è Warning: Using these filters will emit warnings in the console/logs but will not break the PDF generation. The filters are stored in the render tree for future implementation.

üìù Syntax Examples

Single filter:

.element { filter: opacity(0.5); }

Multiple filters:

.element { filter: blur(2px) grayscale(0.5) brightness(1.2); }

Drop shadow:

.element { filter: drop-shadow(5px 5px 10px rgba(0,0,0,0.5)); }

Backdrop filter:

.element { backdrop-filter: blur(10px); }
