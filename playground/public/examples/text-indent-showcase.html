Text Indent Showcase

Quick reference for first-line indentation values. Each block shares the same width so you can compare pixels, percentages, negative hanging indents, and local overrides.

Classic Paragraph

text-indent: 2em; Traveling scribes once relied on geometric first-line offsets to separate thought and tone. Today we bring that same cadence to PDF layout.

Percent Based

text-indent: 18%; When brands publish statements, the opening sentence often slides forward by a proportional amount so the typography scales with the column width.

Hanging Quote

text-indent: -1.5em; “Some indents pull left to keep quotation marks outside the column while preserving a flush text edge.”

Local Override

text-indent cascade Default copy inherits the 32px indent from the parent container to match corporate guidelines. Labels reset indent when we need a flush baseline for inline tags.
