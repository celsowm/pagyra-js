2. Radial Gradients

Implementation Notes

  • Linear Gradients: Support for x1, y1, x2, y2 coordinates and gradientUnits
  • Radial Gradients: Support for cx, cy, r, fx, fy with focal point handling
  • Gradient Stops: Full support for offset, stop-color, and stop-opacity
  • Spread Methods: Support for pad, reflect, and repeat methods
  • Units: Both objectBoundingBox and userSpaceOnUse coordinate systems

Note: Gradient rendering would require additional renderer updates, but the parsing and data structures are fully implemented.
